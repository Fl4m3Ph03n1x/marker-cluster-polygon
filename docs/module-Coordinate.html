Module: Coordinate

Creates a Coordinate object with the given latitude and longitude. This object is a useful abstraction as it encapsulates a requirement by the google.maps.polygon that requires an object with this format in order to draw a Polygon.
Source:

See:

Example

const myCoord  = coordinateFactory(0, 1);
 console.log(`Coord is: ${myCoord.lat}, ${myCoord.lng}`);
